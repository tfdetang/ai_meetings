## 文档组织原则

本项目遵循以下文档组织原则:

1. **用户优先** - 用户文档放在根目录,易于访问
2. **简洁明了** - 避免冗余,每个主题只有一个权威文档
3. **分层组织** - 快速开始 → 详细指南 → 故障排除
4. **开发者分离** - 开发文档放在专门目录
