## 参与者列表

```
参与者: [● Alice (产品经理)] [● Bob (工程师)] [● Carol (设计师)]
        ↑ 蓝色圆点          ↑ 紫色圆点      ↑ 绿色圆点
```

每个标签包含：
- 彩色圆点指示器
- 代理名称
- 角色名称
- 彩色边框和背景

## 快速 @ 标签

```
快速 @: [@Alice] [@Bob] [@Carol]
        ↑ 蓝色   ↑ 紫色  ↑ 绿色
```

点击标签自动插入 `@代理名 ` 到输入框。

## 设计细节

### 1. 消息气泡

```css
.message-bubble {
  padding: 12px;
  background: [代理颜色背景];
  border-left: 4px solid [代理颜色边框];
  border-radius: 4px;
  margin-bottom: 16px;
}
```

### 2. 标签样式

```css
.agent-tag {
  color: [标签颜色];
  border-left: 3px solid [边框颜色];
  padding-left: 8px;
}

.agent-tag::before {
  content: "●";
  color: [边框颜色];
  margin-right: 6px;
}
```

### 3. 圆点指示器

```css
.color-indicator {
  display: inline-block;
  width: 8px;
  height: 8px;
  border-radius: 50%;
  background-color: [边框颜色];
  margin-right: 6px;
}
```

## 响应式设计

### 桌面端（> 768px）

- 消息气泡宽度：100%
- 左边框：4px
- 标签大小：正常
- 圆点大小：8px

### 移动端（≤ 768px）

- 消息气泡宽度：100%
- 左边框：3px
- 标签大小：小号
- 圆点大小：6px
- 字体大小：略小

## 可访问性

### 颜色对比度

所有颜色组合都符合 WCAG AA 标准：
- 文字与背景对比度 ≥ 4.5:1
- 边框与背景对比度 ≥ 3:1

### 辅助识别

除了颜色，还提供：
- 发言者名称标签
- 时间戳
- 轮次编号
- 左边框（形状辅助）

### 色盲友好

- 使用多种视觉元素（颜色 + 形状 + 文字）
- 避免仅依赖颜色传达信息
- 计划添加专门的色盲模式

## 最佳实践

### 1. 创建会议

- 选择 3-5 个代理（颜色最易区分）
- 选择不同角色（更容易识别）

- 参考参与者列表
